# Beleggingsnieuws Tracker

PWA voor beleggingsnieuws per categorie met sentiment.

## GitHub Pages deployen

1. Maak een nieuwe repo op GitHub (bijv. `nieuws-tracker`)
2. Push alle bestanden naar de `main` branch
3. Ga naar **Settings** → **Pages**
4. Bij **Source**: kies **Deploy from a branch**
5. Branch: `main`, folder: `/ (root)`
6. Klik **Save**

## Installeren op Android

Open de URL in Chrome → Menu (⋮) → **App installeren**
